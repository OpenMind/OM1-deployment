# OM1 deployment

This repository contains Docker Compose configurations for deploying the OM1 robot system. It includes services for the main OM1 application, video processing, and the orchestrator.

# Port Usage

- 8000: Riva NMT Local TTS/ASR API
- 8001: Riva ASR Local API
- 8002: Riva TTS Local API
- 8860: Qwen 30B Quantized API
- 8880: Kokoro TTS API
- 8888: Riva AI Orchestrator API
- 50000: Riva Server API
- 50051: Riva NMT Remote TTS/ASR API
